    Redis 数据类型（二）

    2020-06-23

    Redis 数据类型（二）

    无序集合

    Redis 集合（Set）是一个无序的字符串集合。她可以以在 O(1)的的时间复杂度完成添加、删除以及测试元素是否存在。同时 Redis 集合还提供一些服务端的命令从现有的集合出发去进行集合运算，因此你可以在非常短的时间内进行合并（unions），求交集（intersections），找出不同的元素（differences of sets）。
